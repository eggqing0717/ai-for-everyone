# Appendix B: Recommended Learning Resources

Finishing this book is only the starting point. This section helps you find a **reliable next stop**—from free courses to classic great books, from quality videos to open-source tools.

> Notes:
> - **Chinese-friendly** means there is Chinese content, Chinese subtitles, or Chinese community support.
> - The links are all official general URLs. The AI field updates fast, so for specific courses/versions please refer to the official sites.
> - We suggest a step-by-step order of "popular-science videos first → then systematic courses → then hands-on tools."

## Learning Tips for Complete Beginners

1. **Don't be greedy.** Picking one course from above (Andrew Ng or Hung-yi Lee recommended) and finishing it steadily beats bookmarking a pile you never open.
2. **Feel first, reason later.** Watch 3Blue1Brown's animations first to build intuition, then tackle the formulas—you'll get twice the result with half the effort.
3. **You must get hands-on.** Open [Google Colab](https://colab.research.google.com) and run your first piece of AI code for free.
4. **Put it to use.** Using large models like ChatGPT and Ernie Bot to help with your work every day is itself the best kind of learning.
