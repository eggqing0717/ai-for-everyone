# 第四部分：从词向量到 Transformer · 本部分导读

> 如果说前面三部分是在打地基，那么这一部分，我们就要亲手搭起如今所有大模型（ChatGPT、文心一言、DeepSeek……）赖以生存的"承重墙"了。

## 为什么这一部分最重要

你有没有想过一个问题：**ChatGPT 凭什么能听懂你说的话，还能对答如流？**

它既不是真的"懂"中文，也没有一颗人类的大脑。它能做到这一切，靠的是一套叫做 **Transformer** 的技术架构。可以毫不夸张地说：

> **没有 Transformer，就没有今天的大模型时代。**

2017 年，谷歌的几位研究员发表了一篇名叫《Attention Is All You Need》（《你只需要注意力》）的论文。这篇论文提出的 Transformer 架构，像一颗投入湖面的石子，激起了一整个时代的浪潮——后来的 GPT、BERT、以及你听说过的几乎所有大模型，全都是它的"徒子徒孙"。

所以，这一部分是**全书的核心，也是最烧脑的一段路**。但请你放心，我会用最通俗的比喻、最慢的节奏，一步一步带你走。走完这段路，你会有一种"啊，原来是这么回事"的通透感。

## 阅读建议

- **别跳章。** 这一部分环环相扣，前一章是后一章的钥匙。跳着读很容易迷路。
- **慢慢来。** 遇到卡壳的地方，停下来，多读一遍那个比喻。理解比速度重要。
- **不必记公式。** 本书几乎不用数学公式；万一出现了极简的式子，我保证立刻用大白话给你翻译一遍。
- **第18章可以读两遍。** 它是全书的招牌，第一遍先建立整体印象，第二遍再抠细节，效果最好。

准备好了吗？我们先从一个最基本的问题出发：**电脑根本不认识字，那它到底是怎么处理语言的？** 翻开第15章，我们开始。
