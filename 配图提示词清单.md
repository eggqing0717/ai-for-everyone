# 全书配图 · 生图提示词清单

> 本清单从《AI for Everyone / 人人都能懂的 AI》全部章节中抽取，汇总了每一处**图片占位**及其对应的**中文生图提示词**，供你用生图 AI 批量出图后一次性替换回正文。

## 一、使用说明

1. **图片放置位置**：所有正文里的图片都引用自 `assets/images/` 目录（正文写法形如 `![替代文字](../assets/images/文件名.png)`）。请把生成好的图片，**严格按本清单的"目标文件名"命名**，放入项目根目录下的 `assets/images/` 文件夹即可，正文无需任何改动就能自动显示。
2. **文件名务必一致**：正文里的引用路径是写死的，文件名（含大小写、连字符、`.png` 后缀）必须与本清单**完全一致**，否则图片无法显示。
3. **格式建议**：优先输出 **PNG**（透明或白底皆可）；若为纯矢量结构图（如流程图、架构图），也可用 **SVG**，但需同步把正文引用后缀改为 `.svg`（本清单默认 `.png`，若不改后缀请仍导出 PNG）。
4. **统一风格**：全书插画建议保持一致的视觉语言——**扁平化科技插画风、蓝紫为主色调、配以中文标注**，这样替换后整本书观感统一。详见文末"批量出图建议"。

### 批量出图建议

1. **统一画风**：全书采用**扁平化科技插画风**，避免写实或 3D 渲染（个别"3D 山谷""烧水"类可按提示词灵活处理，但整体线条与质感保持一致）。
2. **统一配色**：以**蓝紫为主色调**；对比类图可用"绿色=正面/机遇、橙红=负面/风险"作辅色；附录两张已给出精确色值，可作为全书色板参考——科技蓝 `#2E7BF6`、青绿 `#22C1A6`、橙色高亮 `#FF8A3D`，背景浅灰白。
3. **中文标注**：所有图内文字**一律使用中文**（英文术语如 GPT、CNN、Q/K/V、Σ 可保留），字体清晰、字号适中，确保缩小后仍可读。
4. **分辨率建议**：正文横向图建议 **1600×900 或 1920×1080（16:9）**；方形/热力图类建议 **1200×1200（1:1）**；导出 **PNG（72–150 DPI 足够网页显示）**。若为结构/流程图，优先考虑 **SVG** 以保证缩放清晰（注意同步修改正文引用后缀）。
5. **命名务必与表格一致**：生成后按本清单"目标文件名"逐一重命名，统一放入 `assets/images/`，即可无缝替换全书占位图。
