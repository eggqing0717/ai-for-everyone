# 附录A AI 常用术语速查表

这是一份"AI 名词字典"。每一条都先给你**一句大白话**（读得懂就行），再补一句**稍正式的解释**（想深入时看）。遇到不懂的词，回来查一查即可，不用背。

> 阅读方法：括号里是英文原词，方便你在英文资料里对上号。带 ⭐ 的是全书最高频、最值得记住的核心词。

---

## 一、模型相关

**人工智能（AI, Artificial Intelligence）⭐**
- 大白话：让机器像人一样"聪明"地干活。
- 正式说：让机器具备感知、学习、推理、决策等类人智能能力的技术总称。

**机器学习（Machine Learning, ML）⭐**
- 大白话：不靠人写死规则，让电脑自己从大量例子里"看出门道"。
- 正式说：AI 的核心分支，让计算机从数据中自动学习规律并做出预测。

**深度学习（Deep Learning, DL）⭐**
- 大白话：用很多层的神经网络来学习，是机器学习里最能打的一派。
- 正式说：基于多层神经网络的机器学习方法，能自动从数据中提取多层次特征。

**模型（Model）⭐**
- 大白话：机器学到的那套"本领"，输入问题就能给你答案。
- 正式说：通过训练得到的、能把输入映射为输出的数学函数或参数集合。

**算法（Algorithm）**
- 大白话：解决问题的一套固定步骤，像做菜的菜谱。
- 正式说：为完成特定任务而设计的一系列明确、有限的计算步骤。

**参数（Parameter）**
- 大白话：模型内部一堆可以调的"旋钮"，训练就是把旋钮调到最合适。
- 正式说：模型中通过训练学习得到的内部数值（如权重和偏置）。

**权重（Weight）**
- 大白话：表示某个输入"有多重要"的数字。
- 正式说：神经网络连接上的系数，决定输入信号对输出的影响强度。

**特征（Feature）**
- 大白话：用来描述一个东西的关键信息，比如判断房价看"面积、地段"。
- 正式说：从原始数据中提取、用于模型学习和判断的属性或变量。

**分类（Classification）**
- 大白话：判断"这是猫还是狗"这类分门别类的任务。
- 正式说：预测样本所属离散类别的监督学习任务。

**回归（Regression）**
- 大白话：预测一个具体数值，比如"明天几度""这房多少钱"。
- 正式说：预测连续数值输出的监督学习任务。

**聚类（Clustering）**
- 大白话：没人告诉答案，机器自己把相似的东西归成一堆堆。
- 正式说：无监督学习中把相似样本自动分组的方法。

**监督学习（Supervised Learning）**
- 大白话：拿"带标准答案"的题给机器练，像有老师批改。
- 正式说：使用带标签数据进行训练的学习方式。

**无监督学习（Unsupervised Learning）**
- 大白话：只给数据不给答案，让机器自己找规律。
- 正式说：从无标签数据中发现内在结构或模式的学习方式。

**强化学习（Reinforcement Learning, RL）**
- 大白话：像训练小狗，做对了给奖励，做错了没糖吃，慢慢学会最优做法。
- 正式说：智能体通过与环境交互、根据奖励信号优化决策策略的学习范式。

## 二、神经网络相关

**神经网络（Neural Network）⭐**
- 大白话：模仿大脑神经元连接方式搭起来的一张"计算网"。
- 正式说：由大量互联的人工神经元分层组成、可学习复杂映射的计算模型。

**神经元（Neuron / Node）**
- 大白话：神经网络里最小的"计算小单元"，负责接收、加工、传出信号。
- 正式说：网络的基本单元，对加权输入求和后经激活函数产生输出。

**激活函数（Activation Function）**
- 大白话：给神经元装的"开关"，决定信号要不要往下传、传多少。
- 正式说：引入非线性的函数（如 ReLU、Sigmoid），使网络能拟合复杂关系。

**隐藏层（Hidden Layer）**
- 大白话：输入和输出之间那些"幕后干活"的层。
- 正式说：位于输入层与输出层之间、负责特征变换的中间层。

**卷积神经网络（CNN, Convolutional Neural Network）⭐**
- 大白话：最擅长"看图"的网络，认脸、识物全靠它。
- 正式说：利用卷积操作提取图像局部特征、擅长计算机视觉任务的网络。

**循环神经网络（RNN, Recurrent Neural Network）**
- 大白话：擅长处理"有先后顺序"的东西，比如句子、语音。
- 正式说：带有循环连接、能处理序列数据并保留时序信息的网络。

**LSTM（长短期记忆网络）**
- 大白话：RNN 的升级版，记性更好，不容易"忘了前文"。
- 正式说：通过门控机制缓解长序列梯度消失、擅长长期依赖的 RNN 变体。

**参数量（Parameters Count）**
- 大白话：模型"脑容量"的大小，参数越多通常能力越强也越吃资源。
- 正式说：模型中可训练参数的总数，常用于衡量模型规模（如 70 亿参数）。

## 三、数据相关

**数据集（Dataset）⭐**
- 大白话：喂给机器学习的一大堆例子。
- 正式说：用于训练、验证或测试模型的样本数据集合。

**训练集 / 验证集 / 测试集（Train / Validation / Test Set）**
- 大白话：练习题、模拟考、正式考——分开用才知道真本事。
- 正式说：分别用于训练模型、调参选优、最终评估泛化能力的数据划分。

**标签（Label）**
- 大白话：给数据贴的"标准答案"，比如这张图标注"猫"。
- 正式说：监督学习中样本对应的目标输出值。

**标注（Annotation / Labeling）**
- 大白话：人工给数据贴标签的活儿，又累又重要。
- 正式说：为原始数据人工添加标签或标记的过程。

**特征工程（Feature Engineering）**
- 大白话：把原始数据加工成机器更好懂的形式。
- 正式说：对数据进行选择、变换、构造特征以提升模型效果的过程。

**归一化 / 标准化（Normalization / Standardization）**
- 大白话：把数值调到差不多的"量级"，免得大数欺负小数。
- 正式说：将数据缩放到统一范围或分布，加速训练、稳定收敛。

**数据清洗（Data Cleaning）**
- 大白话：把脏的、错的、重复的数据挑出来扔掉或修好。
- 正式说：识别并处理缺失、异常、重复等问题以提升数据质量。

**Token（词元）⭐**
- 大白话：大模型眼里的"字块"，一句话会被切成一个个 token 处理。
- 正式说：文本被切分后的最小处理单位，可能是词、子词或字符。

## 四、训练与优化

**训练（Training）⭐**
- 大白话：让模型反复做题、不断改错，越来越准的过程。
- 正式说：通过优化算法调整模型参数以最小化损失的过程。

**损失函数（Loss Function）⭐**
- 大白话：衡量模型"错得有多离谱"的尺子，越小越好。
- 正式说：度量模型预测值与真实值差距的函数，是优化的目标。

**梯度下降（Gradient Descent）⭐**
- 大白话：像蒙眼下山，每次朝最陡的下坡方向迈一小步，直到谷底。
- 正式说：沿损失函数负梯度方向迭代更新参数以求最小值的优化方法。

**反向传播（Backpropagation）⭐**
- 大白话：把"错误"从后往前一层层传回去，告诉每个旋钮该怎么调。
- 正式说：利用链式法则从输出层向输入层逐层计算梯度的算法。

**学习率（Learning Rate）**
- 大白话：每次调整的"步子大小"，太大容易迈过头，太小走得慢。
- 正式说：控制参数更新幅度的超参数。

**超参数（Hyperparameter）**
- 大白话：训练前人工设定的"配置项"，比如学习率、层数。
- 正式说：训练前需人工设定、不由数据学习得到的参数。

**过拟合（Overfitting）⭐**
- 大白话：死记硬背练习题，一到真考试就懵——学过头了。
- 正式说：模型在训练集表现好、在新数据上表现差，泛化能力不足。

**欠拟合（Underfitting）**
- 大白话：练得太少，连练习题都做不对。
- 正式说：模型过于简单，未能学到数据规律，训练与测试都差。

**泛化能力（Generalization）**
- 大白话：面对没见过的新题也能答对的本事。
- 正式说：模型在未见数据上的表现能力。

**正则化（Regularization）**
- 大白话：给模型"立规矩"，防止它死记硬背（过拟合）。
- 正式说：通过约束模型复杂度（如 L1/L2、Dropout）提升泛化的技术。

**批次（Batch / Batch Size）**
- 大白话：一次喂给模型多少条数据一起学。
- 正式说：每次参数更新所用的样本数量。

**轮次（Epoch）**
- 大白话：把整个数据集完整学一遍叫一轮。
- 正式说：训练过程中完整遍历一次全部训练数据。

**准确率（Accuracy）**
- 大白话：答对的比例，最直观的成绩单。
- 正式说：预测正确的样本占总样本的比例。

## 五、大模型与应用

**大语言模型（LLM, Large Language Model）⭐**
- 大白话：读了海量文字、特别会"说人话"的超大 AI，比如 ChatGPT 背后的模型。
- 正式说：基于海量文本训练、参数规模巨大、擅长理解与生成语言的模型。
- 补充：本书中提到的"大模型"，若无特别说明，主要指这类以文本为主的大语言模型（LLM）。

**基础模型（Foundation Model）**
- 大白话：像"读完大学、打好通识底子"的通才模型，很多专用 AI 都是拿它来"深造"（微调）出来的。
- 正式说：在大规模、多任务/多模态数据上预训练、具备通用能力、常作为下游微调起点的超大模型。

**Transformer ⭐**
- 大白话：如今几乎所有大模型的"发动机"，靠"注意力"厉害起来的。
- 正式说：2017 年提出、基于自注意力机制的深度学习架构，是现代大模型基石。

**注意力机制（Attention）⭐**
- 大白话：让模型学会"抓重点"，看句子时知道该盯哪个词。
- 正式说：根据相关性给不同输入分配权重、聚焦关键信息的机制。

**自注意力（Self-Attention）**
- 大白话：让句子里的每个词都"看看"其他词，理解彼此关系。
- 正式说：序列内部元素之间相互计算注意力权重的机制。

**编码器（Encoder）**
- 大白话：负责"读懂"输入的那半边。
- 正式说：将输入序列编码为语义表示的模块。

**解码器（Decoder）**
- 大白话：负责"往外说 / 生成"内容的那半边。
- 正式说：根据编码信息逐步生成输出序列的模块。

**词向量 / 词嵌入（Word Embedding）⭐**
- 大白话：把文字变成一串数字，让电脑能算"意思相近"。
- 正式说：将词语映射为稠密向量、使语义相近的词在向量空间中靠近。

**预训练（Pre-training）⭐**
- 大白话：先让模型海量"通读"打好底子，练成"通才"。
- 正式说：在大规模无标注数据上训练出通用能力的第一阶段。

**微调（Fine-tuning）⭐**
- 大白话：在通才基础上再针对具体任务"补课"，练成"专才"。
- 正式说：在预训练模型上用特定任务数据继续训练以适配下游任务。

**GPT（Generative Pre-trained Transformer）⭐**
- 大白话：一类"边读边猜下一个字"练出来的生成式大模型，ChatGPT 的家族。
- 正式说：基于 Transformer 解码器、以生成式预训练为核心的模型系列。

**提示词 / 提示工程（Prompt / Prompt Engineering）⭐**
- 大白话：你对 AI 说的话；把话说好，AI 答得更好。
- 正式说：设计输入指令以引导大模型产生期望输出的方法。

**RAG（检索增强生成，Retrieval-Augmented Generation）⭐**
- 大白话：让 AI 先"查资料"再回答，减少瞎编。
- 正式说：将外部知识检索与生成模型结合，提升回答准确性与时效性。

**幻觉（Hallucination）⭐**
- 大白话：AI 一本正经地胡说八道、编造不存在的事实。
- 正式说：模型生成看似合理但与事实不符的内容的现象。

**涌现能力（Emergent Ability）⭐**
- 大白话：模型大到一定程度，突然会了没专门教过的本事。
- 正式说：随规模增大而在小模型上不具备、大模型上突然出现的能力。

**多模态（Multimodal）**
- 大白话：不止会看文字，还能同时懂图片、声音、视频。
- 正式说：能同时处理和理解多种数据形态（文本、图像、音频等）的能力。

**智能体（Agent）**
- 大白话：能自己规划、调用工具、一步步把任务干完的 AI"打工人"。
- 正式说：能感知环境、自主决策并调用工具执行多步任务的 AI 系统。

**参数微调技术 LoRA（低秩适配）**
- 大白话：只动模型一小部分就能"补课"，省钱又省显存的微调法。
- 正式说：通过低秩矩阵注入少量可训练参数、高效微调大模型的方法。

**推理（Inference）**
- 大白话：模型训练好后，实际拿来"用"、给你答案的过程。
- 正式说：使用已训练模型对新输入进行预测或生成的阶段。
