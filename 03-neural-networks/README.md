# 第三部分：神经网络与深度学习 · 本部分导读

你好，我是 Jay。欢迎来到这本书里最"硬核"、也最激动人心的一部分。

如果说前面两部分我们聊的是"AI 是什么"和"机器怎么学习"，那么从这一部分开始，我们要真正掀开当今 AI 引擎盖，看看里面那台轰鸣的发动机——**神经网络**。今天你听到的所有热词，ChatGPT、人脸识别、自动驾驶、AI 绘画，追根溯源，几乎都建立在这套技术之上。

别担心，我们依然会用"人话"来聊。我保证，读完这一部分，你能对"深度学习到底深在哪、机器怎么看懂图片、怎么读懂一句话"有一个清清楚楚的直觉。

## 这一部分我们会聊什么

- **第9章 神经网络：从大脑得到的灵感**——原来这套技术真的是"抄"我们大脑的作业，我们先搞懂最小的零件：一个人工神经元。
- **第10章 神经网络如何学习：反向传播**——网络答错了怎么办？它像一个侦探一样，从结果倒推、层层追责，然后自我修正。
- **第11章 深度学习为什么"深"**——"深"不是玄学，而是让机器一层一层、从粗到细地理解世界。
- **第12章 卷积神经网络 CNN**——机器是怎么"看懂"一张图的？靠一个会滑动的"放大镜"。
- **第13章 循环神经网络 RNN**——处理文字、语音这类"有先后顺序"的信息，机器需要一点"记性"。
- **第14章 常见深度学习模型速览**——用一张对比表，帮你认识 ResNet、YOLO 这些"江湖名门"。

## 给你的三个小建议

1. **别怕"神经"两个字。** 它听起来像生物课，其实核心就是"加加乘乘再做个判断"，小学数学水平足够。
2. **重比喻，轻公式。** 这部分我几乎不写公式，所有难点我都会先讲一个生活里的场景，再点出它的学名。
3. **按顺序读。** 第9、10章是地基，后面的 CNN、RNN 都是在地基上盖楼。地基稳了，后面会很轻松。
